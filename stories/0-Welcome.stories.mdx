# lyfutils 工具库

主要是在项目开发中，总结一些组件、自定义 hooks、工具函数等。

## Components

- 二维码：QRCode
- 轻提示：Toast

## Hooks

- useList: 查询列表
- useDebounceText: 获取 debounce 文本
- useQuery: 获取查询参数

## 安装

````shell
yarn add @liaoyf/lyfutils
````

## 使用

````jsx
import React from 'react';
import { QRCode } from '@liaoyf/lyfutils';

const Demo = () => {
    return (
        <QRCode value="http://www.baidu.com" alt="百度" />
    )
};
````
