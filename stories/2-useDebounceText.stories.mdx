# useDebounceText

<Preview>

  {() => {
       const { targetValue, originValue, onOriginChange, loading } = useDebounceText('');
       return (
           <div>
               <input type="text" placeholder="请输入" value={originValue} onChange={e => onOriginChange(e.target.value)} />

               </div>
           </div>
       )
   }}

## API

````jsx
useDebounceText(initValue: string|undefined, ms: number)
````

***Params***

- ***initValue***：初始化值
- ***ms***：防抖时间段，单位毫秒

***Returns***

- ***targetValue***: 最终的值
- ***originValue***: 原始输入值
- ***onOriginChange***: 原始值改变回调
- ***loading***: 加载中
